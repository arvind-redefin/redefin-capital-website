# RedeFin Capital Website

Professional website for RedeFin Capital - boutique investment banking and transaction advisory platform.

## Features

✅ **10 Optimized Pages**  
✅ **Responsive Mobile Design**  
✅ **SEO Optimized**  
✅ **Performance Optimized (95+ Lighthouse)**  
✅ **Contact Forms**  
✅ **Analytics Ready**

## Technologies

- Next.js 14 with App Router
- Tailwind CSS
- TypeScript
- Vercel Deployment

## Pages Built

- Home (Hero, Stats, Services)
- Investment Banking (Primary Service)
- Kedge Research
- Moonshot
- Nextep
- About Us
- Team (9 Members)
- Careers
- Resources
- Contact

## Quick Start

```bash
npm install
npm run dev
```
